# Jogo Par ou Ímpar

Aplicação desenvolvida em Python que simula o jogo Par ou Ímpar contra o computador.

## Objetivo

Praticar estruturas de repetição, validação de entrada e controle de fluxo por meio da implementação de um jogo interativo.

## Funcionalidades

- Geração de número aleatório pelo computador
- Escolha entre par ou ímpar
- Verificação automática do resultado
- Contagem de vitórias consecutivas
- Encerramento do jogo ao perder

## Conceitos Aplicados

- Estruturas de repetição
- Condicionais encadeadas
- Geração de números aleatórios
- Operadores matemáticos
- Validação de entrada

## Tecnologias

- Python 3
